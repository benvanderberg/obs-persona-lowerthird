Persona Title - Lower Third

Below is a generator to create a lower-third graphic for OBS which shows the persona in which you are presenting a demo. This is ideal if you are using OBS for recording or presenting a demo and need to show different personas.


Step 1: Set Text

Set the text for your graphic by choosing the persona image and setting the header and subhead text.






Step 2: Copy Code

Click the button below to copy the URL to your clipboard.


Step 3: Add to OBS

In OBS, create a Browser Source by doing the following:

  1. In the Source panel, click + and select Browser.
  2. Name your Browser source and click OK.
  3. Paste the URL into the URL field.
  4. Set Width field to 800.
  5. Set the Height field to 300.
  6. Check Shutdown source when not visible..
  7. Check Refresh browser when scene becomes active.
  8. Click OK.
